Pheromon

internet of things / data-science
CHALLENGE

How to capture data everywhere, at any time, with different channels ?

SOLUTION

We developed a technology of devices able to communicate with a centralized platform by cellular network.
You can plug any kind of sensors on these devices (meteo, camera, bluetooth, …), and will push their measurements by remote access. You can administrate the swarm of devices with a dedicated dashboard.
Best advantage: you can embed machine learning inside and update it remotely.

POTENTIAL

Along our different projects, we installed several devices on places devoid of internet connection (recycling centers, beaches).
The technology is generic, unlimited on the activities it could be used for, or the diversity of sensors to plug in. Furthermore, we developed a dedicated API to integrate the data on a foreign system.
This project is open-source, you can find the source code on Github.
